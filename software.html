Number System Conversion Game

A game where players can practice number conversions from the following number systems: binary, octal, decimal, and hexadecimal

See More

Originally created for the Create-Performance-Tasks for the AP Computer Science Principles class, the program was created using Java with the built-in JSwing UI library. The player can choose the number system they want to be given and the number system they want to convert to (for example, binary to decimal), the difficulty, and the number of questions they would like to answer. Once the game begins, the player is greeted to a Kahoot like iterface where the user selects the correct number conversion.

Geolocator Using Antenna Triangulation


Finding the location of a device using the triangulation of antenna signals from three different devices transmitting the signal.

See More

The project utilizes the Scapy Python library, a Raspberry Pi, and 3 Wi-Fi range extenders transmitting a 2.4GHz signal. Currently, the 3 Wi-Fi rangee extenders are placed 10ft apart from each other to create a triangular area where location of the Raspberry Pi can be detected through a program running on the Rpasberry Pi. However, the final goal of the project would be reverse the roles. The Raspberry Pi will have 3+ antennas receiving signals and a single transmitting antenna in the middle of those three antennas. The goal would be detect other Raspberry Pis through the singal that it's transmitting using the 3+ receiving antennas.

Music Sheet Image to MIDI File Converter (Under Active Development)


A program that is able to take a picture of a music sheet and convert it to a MIDI file so that the user can hear what the music piece sounds like before they play it.

See More

The program is going to utilize the PyTorch machine learning library to train a model that is capable of detecting the different legends on the music sheet—the notes, the key changes, the timing, etc.—and inform the computer about it. Once it's done that, it takes the data that it's detected and use it to create the MIDI file to play the piece back to the user.
